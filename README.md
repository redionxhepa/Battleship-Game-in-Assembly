# Battleship-Game-in-Assembly

This is the implementation of the Battleship game in Intel 8052 written in Assembly language. The output is shown in a big LCD screen and the screen is divided into half,each half for a particular player.The input is retrieved using a keypad.Each player has its own turn,has one side of the screen and the input is fed to the device through the same keypad. The full implementation is at the Battleship.asm file.
This project was done for the Microprocessors course in collaboration with Levent Gungen. It received the maximal points (100) and esteemed as one of the best projects among 150 students. We were the first team in our class to be able to use a big LCD screen using Intel 8052.
